# データモデル

本システム全体で使用するデータベーススキーマ。

## テーブル一覧

- `raw_materials`: 外部情報源から取得した素材
- `drafts`: 投稿ドラフト
- `posts`: 実際に投稿した投稿記録
- `post_metrics`: 投稿の反応データ（時系列）
- `accounts`: 運用対象アカウント定義
- `info_sources`: 情報源定義

## raw_materials

外部情報源から取得した素材を保存する。

### 主要カラム
- 一意識別子
- 対象アカウント（accounts への参照）
- 情報源（info_sources への参照）
- 取得日時
- 原データ（JSON）
- 正規化されたタイトル
- 正規化されたURL
- 正規化された概要
- 正規化されたメタデータ（JSON）
- 処理ステータス（未処理 / 処理済 / 選別除外 / エラー）

## drafts

post-generator が生成した投稿ドラフトを保存する。

### 主要カラム
- 一意識別子
- 対象アカウント
- 元素材（raw_materials への参照）
- 使用した型（T1〜T5）
- 投稿本文
- 添付メディア候補（JSON）
- 生成日時
- 投稿予定時刻
- レビューステータス（未レビュー / 承認 / 却下）
- レビュー日時

## posts

実際にXに投稿された投稿の記録。

### 主要カラム
- 一意識別子
- 対象アカウント
- 元ドラフト（drafts への参照）
- 使用した型
- 投稿本文
- 投稿URL
- 投稿予定時刻（drafts から引き継ぎ）
- 実投稿日時
- 投稿記録日時

## post_metrics

投稿ごとの反応データを時系列で保存する。

### 主要カラム
- 一意識別子
- 対象投稿（posts への参照）
- 計測日時
- インプレッション数
- いいね数
- リポスト数
- リプライ数
- ブックマーク数
- プロフィールクリック数
- データ取得元（手動入力 / X API 等）

## accounts

運用対象アカウントの定義。

### 主要カラム
- 一意識別子
- アカウント名
- ハンドル
- 領域識別子
- プロフィール文
- 有効フラグ

## info_sources

情報源の定義。

### 主要カラム
- 一意識別子
- 情報源名
- 情報源タイプ（API / RSS等）
- 対象アカウント
- 有効フラグ
