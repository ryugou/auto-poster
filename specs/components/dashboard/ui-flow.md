# UI フロー

dashboard の画面構成と遷移。

## 画面一覧

### トップ画面
- アカウントごとのサマリー表示
- 直近の投稿数、合計インプレッション、フォロワー推移

### 投稿一覧画面
- アカウント選択
- 期間・型・ステータスでフィルタ
- 投稿の一覧

### 投稿詳細画面
- 個別投稿の本文、型、投稿日時
- メトリクスの時系列グラフ
- 元素材・元ドラフトへのリンク

### 型別分析画面
- アカウントごとの型別パフォーマンス
- グラフ等での可視化

### メトリクス入力画面
- 手動入力フェーズ向け
- 投稿を選択し、メトリクスを入力

### ドラフト一覧画面（オプション）
- 生成されたドラフトの確認
- レビュー状況の把握

## 画面遷移

```
トップ画面
  ├→ 投稿一覧画面 → 投稿詳細画面
  ├→ 型別分析画面
  ├→ メトリクス入力画面
  └→ ドラフト一覧画面
```

## 操作方針

- アカウントは常にトップレベルのフィルタとして表示
- 期間指定はデフォルト直近7日 / 30日 / 全期間等を用意
- グラフは主要指標を見やすく、詳細は要求時に展開
