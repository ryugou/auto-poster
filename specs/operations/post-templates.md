# 投稿型テンプレート

本システムで使用する5つの投稿型を定義する。

各型の構成要素・トーンは現在の運用上の例であり、実運用での反応データに応じて更新される。

## 型の一覧

| ID | 型 | 主な用途 |
|---|---|---|
| T1 | 速報型 | 新機能・新サービスの発表速報 |
| T2 | 驚き型 | 強いフックで注意を引く紹介 |
| T3 | 実用解説型 | 具体的な手順・使い方の提示 |
| T4 | まとめ型 | 数字入りの一覧（「〇選」形式） |
| T5 | インパクト衝撃型 | 感情を揺さぶる大ニュース |

## 型選択の基準

Step4 で素材から型を選ぶ際の指針:

- 新発表 → T1 または T5
- 特定ツール・作品を単体で扱う → T2 または T3
- 複数を並べる → T4
- 大ニュース（アニメ化、メジャーアップデート等） → T5

1素材から複数型を生成することも可能。

## 各型の構造（現在の例）

各型の「構成要素」「トーン」は現運用上の初期例。運用データに応じて更新する前提。

### T1: 速報型（例）

**構成要素**
- 冒頭の【速報】マーク
- 主語 + 動作（誰が何をしたか）
- 箇条書きで要点（できること / 料金 / 公開日）
- 画像またはリンク

**トーン**
事実ベース、短文、情報密度高め。

### T2: 驚き型（例）

**構成要素**
- 冒頭の強いフック（【驚愕】【ヤバい】等）
- 対象名の提示
- 何が驚きか（簡潔に）
- 補足（無料 / 簡単 / 新技術など）
- 画像

**トーン**
感情を前に出す、断定的、短文。

### T3: 実用解説型（例）

**構成要素**
- 冒頭の主張（「〇〇が劇的にラクになる」「知らないと損」等）
- 具体的な手順（2〜3ステップ）
- 効果・結果
- 画像またはリンク

**トーン**
具体性重視、実用性を強調、手順は短く。

### T4: まとめ型（例）

**構成要素**
- 冒頭の見出し（「今週の◯◯5選」「◯◯ランキング」等）
- 番号付きリスト
- 各項目は簡潔な説明
- 補足または誘導
- 画像またはリンク

**トーン**
網羅性・一覧性を強調、数字で具体性を示す。

### T5: インパクト衝撃型（例）

**構成要素**
- 冒頭の強い表現（「なんと」「遂に」等）
- 大きな変化・発表内容
- 含意（これで何が変わるか）
- 推奨アクション
- 画像

**トーン**
感情的、スケール感、社会的・業界的な意味を示唆。

## 共通ルール

### 文字数
- 単発投稿: 140〜270字を目安
- スレッド: 各140〜270字

### 画像・メディア
- できるだけ画像を添付
- 著作権ルールは [../CLAUDE.md](../CLAUDE.md) のNGルール参照

### リンク
- 投稿本文にリンクを入れる場合、Xのアルゴリズム評価低下を考慮
- スレッド投稿の場合、リンクは最後のポストに配置

### 絵文字
- 使用可、ただし過度な装飾は避ける

### ハッシュタグ
- 投稿本文には最小限
- プロフィールに記載されたハッシュタグを優先

## 型と領域の組み合わせ例

### A: AI新着まとめ

| 型 | 使用例 |
|---|---|
| T1 速報型 | 新AIツールのリリース発表 |
| T2 驚き型 | 意外な使い方、強力な機能の紹介 |
| T3 実用解説型 | プロンプト紹介、使い方手順 |
| T4 まとめ型 | 今週話題のAI5選、無料AIツール10選 |
| T5 インパクト衝撃型 | 大手AIサービスの大型アップデート |

### B: マンガ新刊速報

| 型 | 使用例 |
|---|---|
| T1 速報型 | 本日発売の新刊情報 |
| T2 驚き型 | 話題作の発見・紹介 |
| T3 実用解説型 | セール・キャンペーン情報 |
| T4 まとめ型 | 今週の新刊5冊、ランキング |
| T5 インパクト衝撃型 | アニメ化・実写化決定 |
